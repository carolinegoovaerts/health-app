## Development cycle
CLI scripts

### Clean the workspace
```
npm run clean
```

### Install dependencies
```
npm install
```

### Run unit tests
```
npm run test
```

### Lint and fix files
```
npm run lint
```

### Compile and run for development
```
npm run serve
```

### Compile and minify for production
```
npm run build
```
